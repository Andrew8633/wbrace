What is a Board Race?

A board race is a fast-paced classroom activity where students compete in teams to fill the board with correct answers as quickly as possible. It’s commonly used in ESL classrooms for vocabulary review, spelling, grammar, or any recall-based task.

This site is designed to be projected directly onto the whiteboard — either using a standard projector or an interactive whiteboard — so students can race to complete the prompts using markers or touch input.

Create or Edit a Race

Use the builder to create new board race templates or load and update your saved ones. You can store up to 10 sets.

How to Use WB Race

  • Create or load a template with up to 10 prompts or words.
  • Load the template to the race screen to project in class.
  • Students compete to write the answers quickly on the board.
  • You can edit, rename, and reuse any set anytime — from the same computer.
